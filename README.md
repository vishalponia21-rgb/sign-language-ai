# Sign Language AI 🤟

Real-time ASL recognition using MediaPipe + ML

## Features
- 26 alphabets detection
- 96% accuracy
- Live webcam feed
- Word builder

## How to Run
streamlit run app.py

## Tech Stack
- MediaPipe, OpenCV, Scikit-learn, Streamlit

## Results
- Kaggle ASL dataset (10,045 samples)
- Random Forest classifier
